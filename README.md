# Photo Sharing Solution build on Appwrite

<img src="https://raw.githubusercontent.com/mariusbolik/picco/main/public/preview.gif" />

1. Install dependencies
```
yarn
```
2. Create an `.env` file with you appwrite keys
```
APPWRITE_ENDPOINT=...
APPWRITE_PROJECT=...
APPWRITE_API_SECRET=...
```
3. Run the project
```
yarn dev
```
